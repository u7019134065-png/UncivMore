# UncivMore

Мод для Unciv, добавляющий большое количество наций, юнитов и другого контента.

## Установка

Скачайте этот репозиторий и поместите папку `UncivMore` в директорию `mods` вашей версии Unciv.

## Содержимое

- Новые нации
- Новые юниты
- И другое дополнительный контент

## Требования

- Unciv (последняя версия)
